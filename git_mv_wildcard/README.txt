# 功能：del命令实现了将文件或目录移动到回收站的功能，而非直接删除，提供安全删除的操作。
# del命令支持以下功能：
#
# 1. **将文件/目录移动到回收站**：
#    - 该命令通过将指定文件或目录移动到用户主目录下的 `~/Trash` 文件夹来实现回收站的功能。
#    - 如果指定的目标是文件，直接移动；如果目标是目录，且未使用 `-r` 选项，则会提示错误，要求使用 `-r` 选项来递归删除目录。
#
# 2. **选项**：
#    - `-r`：递归处理目录（用于移动目录到回收站）。如果不使用此选项，删除目录时会报错。
#    - `-f`：强制删除，跳过删除前的确认提示，直接执行删除操作。
#    - `--help`：显示帮助信息，列出命令使用说明和支持的选项。
#
# 3. **交互式确认**：
#    - 默认情况下，del命令在删除前会询问用户是否确认删除指定的文件或目录。如果输入 `y`，则进行删除；如果输入 `n`，则取消删除操作。
#    - 使用 `-f` 选项可以跳过这一确认步骤，直接执行删除操作。
#
# 4. **仅输入 `del` 时**：
#    - 当不指定任何文件或目录时，执行 `del` 命令会提示用户使用 `del --help` 来查看使用说明。
#
# 5. **清空回收站**：
#    - `~/Trash/clean` 脚本可用来清空回收站中的所有文件和目录（除 `clean` 脚本本身外）。在执行此操作时，会询问用户是否确认清空回收站。
#
# 6. **例子**：
#    - 将文件 `file.txt` 移动到回收站：
#      del file.txt
#    - 将目录 `mydir` 移动到回收站：
#      del -r mydir
#    - 强制删除文件，无需确认：
#      del -f file.txt
#    - 显示帮助信息：
#      del --help
